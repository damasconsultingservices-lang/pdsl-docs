# Code of Conduct
Version: v0.1.x documentation governance alignment
## Purpose

The Public Diplomacy Simulation Lab (PDSL) documentation repository exists to support disciplined, governance-oriented, theory-constrained research.

This Code of Conduct ensures that discussion and collaboration remain professional, rigorous, and aligned with the project’s scope.

---

## Expected behavior

Participants are expected to:

- Engage respectfully and professionally
- Focus on substantive improvements to clarity and governance
- Critique ideas without attacking individuals
- Maintain academic and analytical integrity
- Ensure all contributions maintain evidentiary discipline and accurately represent the scope and limitations of the framework.

---

## Unacceptable behavior

The following are not permitted:

- Harassment or personal attacks
- Political campaigning unrelated to documentation clarity
- Promotion of operational influence, targeting, or manipulation tactics
- Attempts to reframe PDSL as predictive or causal modeling

---

## Scope alignment

PDSL is a deterministic simulation framework for controlled scenario reasoning. It is not:

- A forecasting engine
- A behavioral manipulation tool
- A political persuasion instrument

Contributions must remain consistent with these boundaries.

---

## Enforcement

The repository owner maintains discretion to remove contributions that violate this Code of Conduct or compromise governance integrity.

---

## Reporting

Concerns may be communicated directly to the repository owner.
