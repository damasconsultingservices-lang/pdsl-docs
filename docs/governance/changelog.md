# Changelog

All notable changes to this documentation repository are recorded here.

This repository publishes review-facing documentation for the Public Diplomacy Simulation Lab (PDSL).
PDSL is a deterministic, theory-constrained simulation framework for controlled scenario reasoning.
It is not a forecasting system and does not perform causal inference.
