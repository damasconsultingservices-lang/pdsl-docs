# Cómo citar

## Cita del repositorio

Cite este repositorio usando el archivo legible por máquina `CITATION.cff` en la raíz del repositorio.

**Recomendado**
- Cite la versión etiquetada: **v0.1.1-review-ready**
- Evite citar commits sin etiqueta para artefactos de revisión

## Guardarraíl de citación

Los resultados de PDSL son construcciones interpretativas para razonamiento controlado por escenarios. Esta versión de documentación no respalda afirmaciones de pronóstico ni inferencia causal.
