# Cómo citar PDSL

## Citación del repositorio

Este repositorio debe citarse utilizando el archivo legible por máquina `CITATION.cff`, ubicado en la raíz del repositorio.

**Recomendación**
- Citar la versión etiquetada: **v0.1.1-review-ready**
- Evitar citar commits no etiquetados cuando se utilicen materiales sujetos a revisión académica

## Cláusula de resguardo para la citación

Los resultados producidos por el Public Diplomacy Simulation Lab (PDSL) constituyen construcciones interpretativas diseñadas para el razonamiento de escenarios controlados.  
Esta versión de la documentación **no** respalda afirmaciones predictivas, inferencia causal ni generalizaciones empíricas.

El uso académico y de políticas públicas debe reconocer explícitamente estas limitaciones metodológicas.
