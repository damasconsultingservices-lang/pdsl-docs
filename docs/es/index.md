# Public Diplomacy Simulation Lab (PDSL)

Bienvenido/a a la documentación pública (orientada a revisión) de **PDSL**.

PDSL es un entorno de simulación determinista y restringido por teoría, diseñado para razonamiento de escenarios, demostración metodológica y revisión académica — **no** para pronóstico, predicción ni inferencia causal.
