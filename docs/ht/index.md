# Public Diplomacy Simulation Lab (PDSL)

Byenveni nan dokimantasyon piblik (pou revizyon) **PDSL**.

PDSL se yon anviwònman simulasyon **detèminis**, ki rete anba règ teyori, pou ede fè rezònman sou senaryo, fè demonstrasyon metodolojik, epi sipòte revizyon akademik — **li pa** pou previzyon, prediksyon, ni pou pwouve koz-ak-efè (enferans kozèl).

- **Politik (kanonik / English):** ../policy/

## Nòt pou revizyon
Rezilta PDSL yo se konstriksyon entèpretatif pou analiz ki kontwole. Pa prezante yo kòm prediksyon oswa kòm prèv kozèl.
